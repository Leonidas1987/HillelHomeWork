Вгадай результат

  1. 'number' + 3 + 3 Результат: number33 --> Пояснення: Операція додавання тут використовується для конкатенації рядків.

  2. null + 3 Результат: 3 --> Пояснення: null при приведенні до числа стає 0.

  3. 5 && "qwerty" Результат: "qwerty" --> Пояснення: Оператор && повертає перший помилковий операнд або останній операнд,якщо всі операнди дійсні.У даному випадку 5 правдивий.

  4. +'40' + +'2' + "hillel" Результат: 42hillel --> Пояснення: Унарний оператор + перетворює рядки '40' та '2' у числа.Додавання 40 + 2 дає 42,а потім конкатенація з "hillel".

  5. '10' - 5 === 6 Результат: false --> Пояснення: Оператор - наводить рядки до числа,тому '10' - 5 дає 5.

  6. true + false Результат: 1 --> Пояснення: true при приведенні до числа стає 1,а false - 0.Додавання 1 + 0 дає 1.

  7. '4px' - 3 Результат: NaN --> Пояснення: Оператор - не може перетворити рядок '4px' на число

  8. '4' - 3 Результат: 1 --> Пояснення: Оператор - наводить рядки до числа.

  9. '6' + 3 ** 0 Результат: '61' --> Пояснення: Оператор** виконує зведення в ступінь.3 ** 0 дорівнює 1.Конкатенація рядка '6' та числа 1 дає '61'.

  10. 12 / '6' Результат: 2 --> Пояснення: Оператор / наводить рядки до числа.

  11. '10' + (5 === 6) Результат: '10false' --> Пояснення: 5 === 6 false. Конкатенація рядка '10' та рядка 'false' (бо false наводиться до рядка) дає '10false'.

  12. null == '' Результат: false --> Пояснення: null і '' (порожній рядок) не рівні при суворому порівнянні.

  13. 3 ** (9 / 3) Результат: 27 --> Пояснення: 9/3 дорівнює 3, і 3**3 дорівнює 27.

  14. !!'false' == !!'true' Результат: true --> Пояснення: Оператор !! наводить значення до булевого типу.Будь-який непустий рядок у JavaScript істина, тому !!'false' і !!'true' обидва рівні true.

  15. 0 || '0' && 1 Результат: 1 --> Пояснення: Оператор && має більший пріоритет,ніж ||.'0' && 1 дорівнює 1 (оскільки обидва операнда істинні). Потім 0 | 1 і 1.

  16. (+null == false) < 1 Результат: false --> Пояснення: +null перетворює null на 0.0 == false і true.true < 1 і false.

  17. false && true || true Результат: true --> Пояснення: false && true дорівнює false.

  18. false && (false || true) Результат: false --> Пояснення: false | true дорівнює true,потім false && true дорівнює false.

  19. (+null == false) < 1 ** 5 Результат: false --> Пояснення: +null перетворює null на 0.0 == false і true. 1 ** 5 дорівнює 1. true < 1 і false.
